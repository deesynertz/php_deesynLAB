# CONFIGURATION

go to ``/database`` then configure connection in db_connection.php file.

```powershell
define('DB_HOST','....');
define('DB_USER','....');
define('DB_PASS','....');
define('DB_NAME','....');
```

## Import

in the same folder you will found ``file_db_new.sql`` create database in your local server and inport db from that file.
